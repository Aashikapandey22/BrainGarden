# 🧠 BrainGarden

> Cultivate ideas, connect knowledge, and grow your second brain.

BrainGarden is a modern digital gardening and personal knowledge management platform inspired by Obsidian. It helps users capture thoughts, organize notes, and build meaningful connections between ideas through an intuitive and interactive interface.

## ✨ Features

* 📝 Create and manage notes
* 🔗 Connect ideas with backlinks
* 🌱 Build a personal knowledge garden
* 📂 Organize content using folders and tags
* 🔍 Fast search and note discovery
* 🌙 Clean and distraction-free interface
* ⚡ Powered by React + Vite

## 🚀 Tech Stack

* React
* Vite
* JavaScript
* CSS
* ESLint

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Aashikapandey22/BrainGarden.git
cd BrainGarden
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## 🏗️ Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
